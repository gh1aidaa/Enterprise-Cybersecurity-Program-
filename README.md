# Enterprise Cybersecurity Program for Saudi Investment Bank (SIB)


A comprehensive cybersecurity program designed for the Saudi Investment Bank (SIB) to enhance security governance, protect critical assets, and ensure compliance with industry standards and Saudi regulations.

---

## Overview

This project presents the design of a comprehensive Cybersecurity Operations Program for the Saudi Investment Bank (SIB).

The program addresses cybersecurity governance, risk management, security policies, incident response, business continuity, disaster recovery, and continuous improvement while aligning security initiatives with business objectives.

---

## Objectives

- Design an enterprise cybersecurity program.
- Identify and classify critical banking assets.
- Develop organizational security policies.
- Align cybersecurity initiatives with business objectives.
- Improve organizational resilience.
- Support regulatory compliance.
- Implement Business Continuity & Disaster Recovery planning.

---

## Project Scope

The project focuses on protecting the following banking services:

- Core Banking Ledger
- Payment Gateway (SWIFT)
- Customer Relationship Management (CRM)
- Mobile & Online Banking
- Internal Audit & Compliance
- Investor Relations Website

---

## Security Frameworks & Standards

This project follows:

- NIST Cybersecurity Framework (CSF 2.0)
- SAMA Cybersecurity Framework
- Personal Data Protection Law (PDPL)
- PCI DSS
